<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>Arbeitsplatzinfo um Switch erweitert - Neue Version des .pdf-Druckers  - Diverse Softwareupdates</h2>

<img src="images/bild_mal1.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>


<li><div><b>Auswertungen: POCSO Netzwerkübersicht um Aktive Komponenten "Switch" erweitert</b><br />
Unter Start - Anwendungen Händler - POCSO Netzwerkübersicht gelangen Sie zur POCSO Netzwerkübersicht, welche um die Information über die in Ihrem Netzwerk eingesetzten 
CISCO Switche erweitert wurde. Dafür wurde ein eigener Reiter im Excel-Sheet angelegt. Beachten Sie, dass auszutauschende Switche (z.B. Cisco 2950 Modelle) rot markiert werden. 
Diese müssen umgehend durch aktuelle Modelle ersetzt werden!

 </div></li>


<li><div>
<b>SW Paket: MAY-Computer eDocPrintPro Vers.3.23.1</b><br />
Neue Version des .pdf Druckers ist ab sofort in der Erstinstallation möglich.
Eine allgemeine Verteilung ist jedoch nicht geplant. Bei eventuellen Problemen kann eine Nachinstallation auf einem vorhandenen PC über den POI Service Desk angefordert werden.
</div></li>

<li><div>
<b>Versionen sonstiger Software im POCSO Standard</b><br />
Wieder einmal werden Security Updates des Oracle Java JRE, Adobe Flash Players sowie Adobe Acrobat Readers verteilt.
</div></li>


</ul>
