<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>Erweiterung Pilotierung Internet Explorer 11 – MIK-BIS Software Update – Adobe Flash Player Vers.15.0.0.223_AG - SW Paket: Oracle Java JRE Vers.7.71_AG 1.0</h2>

<img src="images/bild_mal1.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>




<li><div><b>Erweiterte Pilotierung Internet Explorer 11 bei POCSO</b><br />
Die entsprechenden Vorarbeiten sind abgeschlossen, die Rückmeldungen zur Funktionsfähigkeit mit den in unserem Umfeld eingesetzten Softwareprodukten sind positiv. 
Bis jetzt haben wir in der ersten Phase der Pilotierung ca. 350 Installationen vorgenommen. 
Nun starten wir die 2. Phase der Pilotierung und werden noch im Dezember ca. 700 Anwender bei ausgewählten Betrieben auf IE11 umstellen. 
Das Rollout auf allen Arbeitsplätzen in den POCSO Netzwerken wird bis März 2015 abgeschlossen ein. 


 </div></li>


<li><div>
<b>MIK-BIS Softwareupdate</b><br />

Am 22. und 23.11 wurde ein MIK-BIS Update durchgeführt, der neue Client mittels POCSO SW Verteilung zu den Servern im Betrieb kopiert. 
Der derzeit eingesetzte MIK-BIS Client 5.3.1.0 wird beim ersten Start durch die neue Version 7.0.5.3684 ersetzt.
Die derzeitigen Dokumente (Graphiken, Tabellen, Excel-Reports...) und Funktionen bleiben erhalten.


</div></li>


<li><div>
<b>Software: Adobe Flash Player Vers.15.0.0.223_AG</b><br />

Wieder einmal wird ein Security Update des Adobe Flash Players verteilt.


</div></li>


<li><div>
<b>Software: Oracle Java JRE Vers.7.71_AG 1.0</b><br />

Diese aktuelle Version von Java wird von der Entwicklung in nächster Zeit getestet.
Für Problemfälle, bei denen die neueste Java Version verlangt wird, kann diese dennoch bereits installiert werden.

</div></li>
