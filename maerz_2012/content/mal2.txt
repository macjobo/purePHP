<h1>POCSO: Freigabe Windows 7</h1>
<h2>Windows 7 in der 64-bit-Version ist das neue Standardbetriebssystem im POCSO-Umfeld.</h2>

<img src="images/bild_mal2.jpg" width="190" height="78" title="Windows 7" />



<p>Mit dem Update auf ELSAWin 4.0 ist nun für alle Kernanwendungen die <b>Lauffähigkeit</b> unter Windows 7 sichergestellt. Nach Abschluss des ELSA-Rollouts werden <b>ab April</b> alle neuen Arbeitsplätze mit Windows 7 installiert.</p>

<p>Um leistungsstarke neue Hardware besser nutzen zu können und für die Zukunft gerüstet zu sein, kommt im POCSO-Umfeld die <b>64-bit-Version</b> von Windows 7 zum Einsatz.</p>
<p>
Folgende Informationen sollen zur Orientierung dienen, was den Einsatz von Windows 7 betrifft:</p>


<ul>

<li><div>Eine generelle <b>Migration</b> von Windows-XP-PCs auf Windows 7 ist derzeit nicht notwendig und nicht geplant.
</div></li>

<li><div>Die Neuinstallation bestehender PCs im Problemfall erfolgt für die mit dem Gerät erworbene Lizenz (entweder Windows XP oder Windows 7 - siehe Windows-Aufkleber am Gerät), das heißt, es entstehen für Sie keine zusätzlichen Kosten.
</div></li>

<li><div>Bei <b>individueller Software</b> klären Sie bitte mit Ihrem Lieferanten, ob diese unter Windows 7 in der 64-bit-Version unterstützt wird.
</div></li>

<li><div>Bis Ende 2012 stellen wir bei Geräten mit Windows-7-Lizenz auch eine <b>Neuinstallation von Windows XP</b> zur Verfügung, sollten unter Windows 7 Probleme auftreten.
</div></li>

<li><div>Für Windows XP können wir <b>bis Juli 2014</b> Support leisten - bitte planen Sie die Ablöse Ihrer XP-Geräte bis zu diesem Zeitpunkt ein.
</div></li>

</ul>


<p>Für Fragen zu Windows 7 im POCSO-Netzwerk steht Ihr Key Account Manager gerne zur Verfügung.</p>
