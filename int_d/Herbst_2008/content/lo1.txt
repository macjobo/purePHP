<h1>Client TIPP: Druckerauswahl-Tool</h1>
<h2>Porsche Informatik bietet eine einfache Anwendung für das Hinzufügen von Netzwerkdruckern auf Ihrem Client.</h2>

<p>Das <b>Druckerauswahl-Tool</b> ist schon seit einigen Jahren im Einsatz und vielen von Ihnen sicher schon bekannt. Trotzdem möchten wir an dieser Stelle nochmals auf dieses nützliche Tool hinweisen.</p>

<p>Um einen Netzwerkdrucker auf Ihrem Client hinzuzufügen, wählen Sie unter:
Start - Programme - Eigenentwicklungen - <b>Druckerauswahl</b>:</p>

<p><img src="images/Drucker.bmp" width="354" height="116" alt="Druckerauswahltool" ></p>

<p>Mit der Druckerauswahl können Sie rasch die Netzwerkdrucker in Ihrer Umgebung finden und 
auf einfache Art und Weise Ihrer Arbeitsumgebung hinzufügen. Zudem können Sie über diese Oberfläche auch Ihren <b>Standarddrucker einstellen</b>.</p>
