<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>Neues VAS-Tester Modell 615D - EBV 5.1.01 - Arbeitsplatzinfo um Windows Version erweitert - Lotus Notes Version 9 - Diverse Softwareupdates</h2>

<img src="images/bild_mal1.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>


<li><div><b>VAS Tester: Neues Modell 6150D als POCSO Diagnosetester</b><br />
Ein neues Diagnosetester-Modell, der VAS Tester VAS6150D, wurde vom Werk freigegeben. Somit sind mittlerweile sechs verschiedene VAS-Modelle im POCSO Standard und damit 
in die Windows Domäne integrierbar. 
Die Neuinstallation als POCSO Diagnosetester können Sie anhand
 <a href="http://kb.auto-partner.net/int/hilfe/klbpocso.nsf/AllByID/CAE688EF676778ADC1257C8A0028E2E1?OpenDocument" target="_blank" title="Zur Porsche-Informatik-Infoline">dieser Anleitung&nbsp;&raquo;</a> selbst durchführen. 
Bei Fragen wenden Sie sich an den Service Desk der Porsche Informatik.

 </div></li>


<li><div>
<b>EBV 5.1.01</b><br />
Seit Kurzem läuft das Rollout der neuen EBV-Version 5.1.01., welches noch im März 2016 abgeschlossen sein wird. 
Das Update wird automatisch in der Nacht am Server durchgeführt. Am Tag darauf werden Sie beim Start des Programms auf die 
neue EBV-Version hingewiesen. Diese wird nach Bestätigung Ihrerseits durch eine interne EBV-Funktion automatisch am Arbeitsplatz installiert. 
Eine Anleitung dazu haben wir Ihnen in der Infoline zur Verfügung gestellt:
<a href="http://kb.auto-partner.net/__C1256EBB002981CC.nsf/0/FA0F763A37C57AA1C1257A8500223FE5?Open" target="_blank" title="Zur Porsche-Informatik-Infoline">Dokumentation EBV 5.1.01-Update&nbsp;&raquo;</a>.

Da das Update vollautomatisch durchgeführt wird, erfolgt keine gesonderte Terminvereinbarung. 
Die Updatetätigkeiten der Porsche Informatik sind in den POCSO Dienstleistungen enthalten!

</div></li>


<li><div>
<b>Arbeitsplatzinfo um Windows Version erweitert</b><br />
Unter Start - Arbeitsplatzinfo anzeigen werden die aktuellen Informationen über Ihren Arbeitsplatz angezeigt. 
Dieser Information wurde nun auch die auf Ihrem Arbeitsplatz installierte Betriebssysteminfo hinzugefügt.
</div></li>

<li><div>
<b>Lotus Notes Vers.9.0.1_Rollout im Einzelhandel</b><br />
Das Rollout im Einzelhandel von Lotus Notes 9 beginnt in Kürze.
 Die Porsche Informatik, sowie Teile der Porsche Holding Salzburg, wurden bereits umgestellt. Derzeit erfolgt bei ausgewählten Betrieben die Pilotierung. Vor dem Start des Rollouts werden Sie gesondert informiert.
</div></li>



<li><div>
<b>Versionen sonstiger Software im POCSO Standard</b><br />
Wieder einmal werden Security Updates des Oracle Java JRE Vers.8.73, Adobe Flash Players Vers.20.0.0.306 sowie Adobe Acrobat Readers Vers.11.0.14 verteilt.
</div></li>


</ul>

<p>Alle Fragen rund um die Neuerungen im POCSO-Netzwerk beantwortet Ihnen gerne Ihr Key Account Manager der Porsche Informatik.</p>
