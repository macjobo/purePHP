<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>Neue USV-Software - Verlagerung von Windows-Systemeinhalten - vorgezogener Start der POCSO-Server-Datensicherung - Firmware-Updates - Software-Update für Adobe Flash Player und Shockwave Player - Neue CROSS-Ping-Software</h2>

<img src="images/bild_mal.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>


<li><div>Die <b>Software für Notstromversorgungen</b> wird in den nächsten Wochen auf allen POCSO-Servern auf die neue Version APC Powerchute 9.01 upgedatet. Die neue Version ermöglicht Ihnen den Zugriff auf das USV-Administrations-Programm; damit können Sie USV-Probleme direkt mit dem <b>APC-Support</b> klären und somit die Fehlerlösungszeiten reduzieren. Detaillierte Informationen erhalten Sie nach der Softwareinstallation.</div></li>

<li><div>Um am POCSO-Server für die Images von neuer Software Platz zu schaffen, <b>werden Windows-Systemeinheiten</b> (Pagefile, Drucker-Spooler) vom Laufwerk E:\ auf das Laufwerk D:\ verlegt. Ohne diese Maßnahme könnte bei einigen Betrieben keine neue Software  auf die Server verteilt und damit auch nicht im Netzwerk installiert werden (z.B. in Zukunft Windows 7).</div></li>

<li><div>In den nächsten Wochen werden wir außerdem diverse <b>Firmware-Updates </b>durchführen (Raid Controller, Bios G5/G6, ...), die vom Hersteller für einen reibungslosen Betrieb Ihres POCSO-Servers empfohlen werden.</div></li>

<li><div>Der Beginn der <b>Datensicherung</b> von POCSO-Servern wird von 24:00 Uhr auf 23:00 Uhr vorverlegt, um die immer größeren Datenmengen bis zum Morgen vollständig sichern zu können. 
</div></li>

<li><div>
Aufgrund einer kritischen Sicherheitslücke in den aktuellen Versionen von <b>Adobe Flash Player</b> und <b>Shockwave Player</b> ist ein Update der Programme notwendig. Die allgemeine Verteilung hat bereits begonnen.</div></li>

<li><div>Die aktuell eingesetzte <b>CROSS-Ping-Software</b> wurde auf einen neuen Stand gebracht. Diese Software prüft laufend die Verbindung von Ihrem Netzwerk zum CROSS-Server. Bei Problemen erfolgt automatisch eine Alarmierung des Porsche-Informatik-Leitstandes, um den Fehler schnellstmöglich zu beheben.</div></li>



</ul>

<p>Alle Fragen rund um die Neuerungen im POCSO-Netzwerk beantwortet Ihnen gerne Ihr Key Account Manager der Porsche Informatik.</p>
