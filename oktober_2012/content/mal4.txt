<h1>Hardware: Neuer Farblaserdrucker von Lexmark</h1>

<h2>Mit dem neuen Lexmark C746 haben wir ein Gerät in den POCSO-Standard aufgenommen, das allen Ansprüchen an hochwertigem Fardruck gerecht wird</h2>

<img src="images/bild_mal4.jpg" width="190" height="120" title="Neuer Farblaserdrucker von Lexmark" />


<p>Nun ist es soweit, der erste POCSO Netzwerkserver wurde mit Windows Server 2008 erfolgreich installiert! Ende Oktober erfolgt eine weitere Pilotierung an einem Filialstandort. Danach beginnt die <b>Ablöse der HP ML350G4(p)-Server</b> in den POCSO Netzwerken.</p>

<p>Die <b>neuen Server HP ML350G8</b> werden voll redundant ausgelegt (Raid-5 Festplattenverbund, doppelter Lüfter, Netzteil- und Netzwerkanschluss). Um eine äußerst hohe Verfügbarkeit zu gewährleisten und eine hohe Kosteneffizienz zu erreichen, wurde die Hauptspeicher- und Festplattenausstattung so gewählt, dass eine Serveraufrüstung über die Einsatzdauer von ca. 5-7 Jahren vermieden wird. Darüber hinaus ist dieser Server ein wirklicher <b>Sparmeister beim Stromverbrauch</b>, er beträgt ca. 1/3 des bisherigen Servers.</p>

<p>Betriebe, die einen ML350G4 Server im Einsatz haben, werden von Ihrem zuständigen Key Account Manager der Porsche Informatik auf die Umstellung angesprochen.</p>
