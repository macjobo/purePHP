<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>"POCSO Wireless Lounge" - Betriebssystem Windows Server 2008 - Löschen von überflüssigen ETKA-Datenupdates</h2>

<img src="images/bild_mal.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>

<li><div>
Derzeit arbeiten wir mit <b>"POCSO Wireless Lounge"</b> an einer Lösung, mit der Sie Ihren Kunden über das <a href="http://kb.auto-partner.net/int/hilfe/klbpocso.nsf/f9ca8c1665df61b3c1257345002cdb8b/b6bb0b88c33e3007c12573d10050683d?OpenDocument" target="_blank" title="Zur Porsche-Informatik-Infoline"><b>POCSO-WLAN&nbsp;&raquo;</b></a> ein freies <b>Gäste-WLAN</b> anbieten können. Erste Tests waren sehr erfolgreich. Der Vorteil dieser Lösung liegt darin, dass Sie keine zusätzliche WLAN-Hardware benötigen. Einzig eine eigene ADSL-Datenleitung wird dafür notwendig; eine kostengünstige Lösung dafür arbeiten wir gerade mit A1 Telekom Austria aus. <br />
Mit dieser Lösung können Sie dann auch Tablets (z. B. für Qualitätsmangement, Elektronischen Testbericht NEU, etc.) oder Notebooks, die Sie nicht im POCSO-Netzwerk eingebunden haben, einen <b>Internetzugang</b> ermöglichen. Sobald "POCSO Wireless Lounge" verfügbar ist, werden wir die Lösung im Newsletter detailliert vorstellen.</div></li>

<li><div>
Aktuell laufen die Vorbereitungen zur Umstellung des Server-Betriebssystems von Microsoft Windows Server 2003 auf <b>Microsoft Windows Server 2008 R2</b>. In einem ersten Schritt beginnen wir in den nächsten Wochen damit, die Nachfolgemodelle für die HP-Server ML350 G4 mit dem neuen Betriebssystem auszustatten; diese Geräte haben das Ende ihres Lebenszyklus erreicht.  Die davon betroffenen Betriebe werden gesondert informiert und erhalten ein Angebot zur Umrüstung. </div></li>


<li><div>Die wöchentlichen <b>Datenupdates für ETKA-Clients</b> liegen bekanntlich am POCSO-Windows-Server im Verzeichnis progdat\etka\update. Um nicht unnötig Speicherplatz zu verschwenden, haben wir alle Updates bis zum 1.3.2012 gelöscht. Damit konnten wir pro Windows-Server wieder Platz für fast <b>900 MB</b> schaffen.</div></li>



</ul>



<p>Alle Fragen rund um die Neuerungen im POCSO-Netzwerk beantwortet Ihnen gerne Ihr Key Account Manager der Porsche Informatik.</p>
