<h1>IT Infrastructure: New requirement database</h1>
<h2>To further improve the cooperation between Porsche Informatik Infrastructure services (IS) and the local IT departments, there is now a requirement database for all infrastructure topics.</h2>

<p>The requirement databases have already proved valuable for the Porsche Informatik software products. In order to handle queries from other countries regarding infrastructure topics such as active directory, connections to retailers, planning of a data centre, etc. even more efficiently and effectively in future, we have created the <b>IS Requirement DB</b> for our partner countries.</p>
<p>Access is via your <b>Partner.Net</b> (Porsche Informatik -> Requirement database -> Infrastructure). Requirements can be input under the following categories:</p>

<ul>
<li>Server</li>
<li>Network</li>
<li>Client</li>
<li>Project Management</li>
</ul>

<p>All queries regarding new topics or projects must be authorised by the concerned department (see table). Additionally, the concerned key account manager for the particular country is informed of all queries.</p>
