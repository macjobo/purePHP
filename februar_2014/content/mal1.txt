<h1>POCSO: Neu im POCSO-Netzwerk</h1>
<h2>Änderungen bei der PC Anmeldung - Erweitertes POCSO-Benutzerhandbuch - Domainintegration VAS Geräte - Überarbeitete POCSO-Netzwerkübersicht - Software Update Client-Check.</h2>

<img src="images/bild_mal1.jpg" width="190" height="120" title="Neu im Pocso-Netzwerk" />



<ul>

<li><div>
Sollten noch Windows XP-PCs in Ihrem POCSO Netzwerk in Betrieb sein, wird ab sofort eine <b>wöchentliche Meldung</b> für die Geschäftsführung, 
die Finanzleitung und den EDV-Verantwortlichen eingeblendet, diese auf Windows 7 umzustellen.

</div></li>

<li><div>
Das <b>POCSO-Benutzerhandbuch</b> wurde um einige Punkte erweitert: die Kategorien "Windows XP Ende im POCSO-Netzwerk" und "Anwendungen-Händler" wurden eingefügt und unter der Rubrik zusätzliche Informationen
ist nun eine Beschreibung der POCSO-Netzwerkübersicht zu finden.

Das POCSO-Benutzerhandbuch ist Aufrufbar über Start/Anwendungen - Händler oder direkt über den Desktop.
</div></li>


<li><div>
Auch für die <b>VAS-Tester</b> ist das <b>Ende von Windows XP</b> ein Thema. Vom Werk gibt es mittlerweile eine DVD mit einem Windows 7 Image für eine Reihe von Testermodellen. Unsere Integration der Tester wurde auf Windows 7 angepasst,
die ersten POCSO Diagnosetester mit Windows 7 sind seit Ende Jänner bei Testbetrieben im Einsatz. 
Nach erfolgreicher Pilotierung erfolgt eine Information über die Vorgangsweise bei der Umstellung.
</div></li>

<li><div>
Die <b>POCSO Netzwerkübersicht</b> (Start/Anwendungen - Händler/POCSO Netzwerkübersicht) wurde überarbeitet und bietet nun folgende Zusatzinformationen: Windows XP-Geräte werden rot markiert,
nicht Windows 7-fähige Drucker werden rot markiert, Integration der Windows 7-Fähigkeit von VAS-Testern, neuer Reiter "Drucker Analyse". 


</div></li>

<li><div>
Als stabilitätsverbessernde Maßnahme wurde die Prüfung der Clients auf korrekte Funktion von <b>Symantec Endpoint-Protection</b> und <b>Windows Updates</b> verbessert.
</div></li>

</ul>


<p>Alle Fragen rund um die Neuerungen im POCSO-Netzwerk beantwortet Ihnen gerne Ihr Key Account Manager der Porsche Informatik.</p>
